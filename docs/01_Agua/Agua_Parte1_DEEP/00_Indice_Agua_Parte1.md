# Agua — Parte 1 (Fundamentos Físico‑Químicos)

Este bloque establece los fundamentos **no negociables** del módulo Agua.
Todo lo que sigue (malta, levadura, lúpulo) se apoya en estas bases.

Contiene:
- Qué es realmente el agua cervecera
- Sistemas tampón
- pH vs alcalinidad
- Rol estructural del calcio

Este bloque se usa como **referencia permanente**.
