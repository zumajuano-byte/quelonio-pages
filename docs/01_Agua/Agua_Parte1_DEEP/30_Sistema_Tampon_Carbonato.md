# Sistema tampón carbonato

## 1. Especies químicas
El sistema carbonato incluye:
- ácido carbónico (H₂CO₃)
- bicarbonato (HCO₃⁻)
- carbonato (CO₃²⁻)

La proporción depende del pH.

## 2. Zona relevante para cerveza
El mash ocurre típicamente entre pH 5,2–5,6.
En este rango:
- predomina HCO₃⁻
- pequeñas variaciones generan grandes efectos

## 3. Consecuencia
Este sistema es el principal responsable de:
- resistencia al ajuste de pH
- diferencias regionales históricas
