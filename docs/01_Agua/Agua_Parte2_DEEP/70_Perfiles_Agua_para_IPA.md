# Perfiles de Agua para IPA

## 1. Perfil del agua para NEIPA
- **Sulfato bajo**, **cloro alto**, **calcio moderado**.
- El cloro aporta suavidad al lúpulo y balancea la amargura.

## 2. Perfil para West Coast IPA
- **Sulfato alto** y **bicarbonatos moderados**.
- El sulfato acentúa el amargor, y el pH debe mantenerse entre 5.2 y 5.4.

## 3. Resumen
Asegúrate de tener un balance adecuado de calcio y sulfatos, dependiendo del estilo IPA.
