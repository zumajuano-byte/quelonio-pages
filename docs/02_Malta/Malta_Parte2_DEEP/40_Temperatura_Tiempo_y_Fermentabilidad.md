# Temperatura, tiempo y fermentabilidad

La fermentabilidad se diseña principalmente con el **programa de mash**.

## Rangos típicos
- 62–64 °C → más fermentable
- 66–68 °C → más cuerpo

## Interacción con la malta
Maltas muy modificadas reaccionan más rápido y con menos margen de corrección.

Conclusión:
> La malta define el rango; el mash define el resultado.
