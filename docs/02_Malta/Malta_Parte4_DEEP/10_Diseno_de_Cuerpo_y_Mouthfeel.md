# Diseño de cuerpo y mouthfeel

El cuerpo se diseña principalmente con:
- proporción de extracto no fermentable
- tipo de malta base
- programa de mash (en cruce con Malta Parte 2)

## Herramientas principales
- Maltas base con distinto grado de modificación
- Maltas caramelo en dosis controladas
- Maltas con beta-glucanos (cuando corresponde)

Error común:
> Usar alcohol o lactosa para “corregir” cuerpo.
