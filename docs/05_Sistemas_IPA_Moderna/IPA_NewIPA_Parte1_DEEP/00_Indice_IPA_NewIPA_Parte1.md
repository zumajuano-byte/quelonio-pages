# IPA_NewIPA_Parte1_DEEP — Índice (The New IPA / Scott Janish)

Propósito: capturar aprendizajes **accionables** para diseñar IPA modernas (hazy/juicy) con foco en:
- Tioles (3MH/3MHA/4MMP) y liberación/transformación
- Polifenoles: haziness, “hop bite”/amargor duro, y mitigación
- Variabilidad de lotes de lúpulo y trazabilidad
- Efecto de temperatura de fermentación y arrastre por CO₂

> Nota: Este paquete es “Parte 1” (núcleo aromático + riesgos). En Partes siguientes se amplía a:
> biotransformación de terpenos, timing de dry hop, hop creep/diacetilo, estabilidad/oxígeno, etc.
