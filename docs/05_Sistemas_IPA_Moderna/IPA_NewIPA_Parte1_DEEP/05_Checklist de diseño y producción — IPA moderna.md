# 05_Checklist de diseño y producción — IPA moderna (Parte 1)

## Objetivo sensorial
- Tropical/guayaba/maracuyá (tioles + monoterpenos frutales)
- Amargor suave (minimizar hop bite/astringencia)
- Estabilidad aromática (minimizar stripping y pérdidas)

## Palancas de proceso cubiertas en esta Parte
### A) Tioles
- Definir si queremos empujar 3MH→3MHA o aportar 4MMP (cuidado “catty”).
- Elegir lúpulos/formato considerando ubicación en lupulina y material vegetal.
- Gestionar almacenamiento (frío) y frescura.

### B) Temperatura / stripping
- Diseñar perfil térmico para retener volátiles (especialmente tioles).
- Registrar vigor de fermentación (CO₂) y comparar lotes.

### C) Polifenoles / hop bite
- Estimar riesgo por g/L de DH y tiempo de contacto.
- Si hay “hop burn”, considerar:
  - descanso frío/clarificación
  - (si existe) centrífuga como mitigación rápida

### D) Trazabilidad
- Batch sheet obligatorio: proveedor/cosecha/lote/formato/analítica.
- Correlacionar variabilidad de lotes con desviaciones sensoriales.

## Salidas esperadas (para el Vault)
- Plantilla de “Ficha de Lote de Lúpulo”
- Plantilla de “Batch Record IPA” con:
  - T° real por día
  - timing DH/whirlpool
  - DO/oxígeno (si se mide)
  - sensorial estructurado
