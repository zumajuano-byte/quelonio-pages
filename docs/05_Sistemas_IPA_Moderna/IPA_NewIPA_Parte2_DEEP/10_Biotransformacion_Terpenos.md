# Biotransformación de terpenos

La biotransformación ocurre cuando la levadura:
- libera compuestos aromáticos ligados (glucósidos)
- convierte terpenos en formas más aromáticas

Ejemplos:
- Geraniol → Citronellol
- Linalool → derivados oxigenados

Variables críticas:
- cepa
- fase de fermentación
- temperatura
