# Planificación de dry hopping

- Momentos clave para el dry hopping: fermentación primaria, fermentación secundaria
- ¿Cómo saber cuándo hacer el dry hop?
  - Estabilidad de fermentación (CO₂ activo, pH de fermentación)

Objetivo: maximizar la **extracción aromática** sin sobrecargar el perfil con amargor.
