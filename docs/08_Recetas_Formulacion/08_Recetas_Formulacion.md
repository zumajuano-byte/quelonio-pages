# Recetas y formulación

## Qué es
Este módulo es para diseñar y ajustar recetas sin perderse: qué cambia el cuerpo, el amargor, el aroma, el alcohol y la tomabilidad.

## Para qué sirve
- Ajustar recetas lote a lote con criterio (no “a ojo”).
- Estandarizar cálculos básicos.
- Dejar registro de decisiones y resultados.
